## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Environment Variables

Create a `.env.local` file (optional):

```
GOOGLE_MAPS_API_KEY=your_api_key_here
```

If no API key is provided, the site will use mock data.
